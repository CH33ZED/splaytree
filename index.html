Splay Trees!

Splay Trees: A special type of binary search tree. Space: O(n)
Average run time: O(logn)

Worst case: amortized O(logn), meaning some can take O(n) time to find, although this is rare.

Special Uses: Allows you to access recently used elements quicker.

It “splays” with every search, moving the key you dealt with (or the parent of the key you deal with in case of removal) up toward the root.
